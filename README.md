\# 🚀 CyberAI-Agent  

\### AI Threat Detection + Automated Response System  



CyberAI-Agent is a full-stack AI-powered cybersecurity system that detects anomalous network behavior and automatically generates response actions such as IP blocking, host isolation, and alert generation.



It combines:



\- 🤖 Machine Learning (Anomaly Detection)

\- 🔐 Cybersecurity Threat Modeling

\- ⚙️ Agentic Automated Response

\- 📊 Real-Time Analytics Dashboard

\- 🌐 Full Stack Architecture (Flask + React)



---



\## 🧠 Features



\### ✅ AI-Based Threat Detection

\- Isolation Forest anomaly detection model

\- Heuristic rule engine

\- Anomaly score generation

\- Custom event analyzer



\### ✅ Agentic Response Engine

\- Automatic severity classification (LOW / MEDIUM / HIGH / CRITICAL)

\- Simulated automated actions:

&nbsp; - BLOCK\_IP

&nbsp; - ISOLATE\_HOST

&nbsp; - ALERT

\- Action logging system



\### ✅ Live Threat Analytics Dashboard

\- Real-time anomaly score trend

\- Severity distribution visualization

\- Interactive custom threat simulation

\- Logs viewer



\### ✅ Model Retraining (Agentic Learning)

\- Adjustable contamination parameter

\- Adjustable number of estimators

\- Baseline traffic simulation

\- On-demand retraining via API
